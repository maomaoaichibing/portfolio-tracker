# 持仓智投 - AI投资监控助手

面向二级市场投资者（港美A股），通过AI分析持仓截图，自动生成监控体系，帮助投资者跟踪持仓逻辑变化。

## 核心功能

1. **持仓识别** - 上传持仓截图，AI自动识别标的、数量、成本
2. **智能分析** - AI分析每只股票过去一年的涨跌逻辑
3. **监控体系** - 自动生成需要监控的关键指标
4. **逻辑跟踪** - 持续跟踪涨跌逻辑是否发生变化，及时提醒

## 技术栈

- 前端: HTML + Tailwind CSS + 原生 JavaScript
- 后端: Node.js + Express
- 数据库: SQLite
- AI: Kimi API (截图识别 + 逻辑分析)

## 快速开始

```bash
# 安装依赖
cd portfolio-tracker
npm install

# 启动服务
npm start

# 开发模式
npm run dev
```

访问 http://localhost:3000

## 核心监控逻辑

系统会针对每只持仓股票分析：

1. **过去一年的涨跌驱动因素**
   - 宏观环境变化
   - 行业景气度
   - 公司基本面
   - 市场情绪

2. **生成监控指标**
   - 估值指标 (PE/PB)
   - 业绩指标 (营收/利润增速)
   - 行业指标 (政策/竞争格局)
   - 市场指标 (资金流向/情绪)

3. **持续跟踪**
   - 定期检查指标变化
   - 识别逻辑变化信号
   - 推送重要提醒
